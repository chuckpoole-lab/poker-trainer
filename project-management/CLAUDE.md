# Poker Trainer — Project Instructions

When starting a new session on this project:

1. Read WORK-PLAN.md, DASHBOARD.md, and the most recent work summary file to understand current status
2. Set up a TodoList showing open tasks from the work plan
3. Give me a brief recap: what was accomplished last session and what's next up
4. Ask if I want to pick up where we left off or work on something else

## Key files
- **WORK-PLAN.md** — current priorities and open tasks
- **PROJECT-PLAN.md** — full roadmap and strategy
- **DEVLOG.md** — development history
- **DASHBOARD.md** — quick reference links and stats

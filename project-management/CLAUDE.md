# Poker Trainer — Project Instructions

When starting a new session on this project:

1. Read SESSION-LOG.md first — it has what happened last time and what's next
2. Read WORK-PLAN.md for the full priority list and open tasks
3. Set up a TodoList showing open tasks
4. Give Chris a brief recap: what was accomplished last session and what's next up
5. Ask if he wants to pick up where we left off or work on something else

## Session rules
- **At end of every session:** Update SESSION-LOG.md with what was done, decisions made, and what's next
- **If status or progress changed:** Update projects_data.json in the cowork_projects folder
- **If switching to another project:** Update this session log first before jumping

## Key files
- **SESSION-LOG.md** — session-by-session history (read this first)
- **WORK-PLAN.md** — current priorities and open tasks
- **PROJECT-PLAN.md** — full roadmap and strategy
- **DEVLOG.md** — development history
- **DASHBOARD.md** — quick reference links and stats
